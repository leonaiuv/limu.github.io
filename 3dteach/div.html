2. DIV的定位

静态定位 (static)
相对定位 (relative)
绝对定位 (absolute)
固定定位 (fixed)

通过position属性，可以精确控制div的位置：
• static：默认定位，按照文档流排列
• relative：相对定位，相对于原来的位置
• absolute：绝对定位，相对于最近的定位祖先元素
• fixed：固定定位，相对于浏览器窗口

3. DIV的常用属性

.example-div {
  width: 300px;
  height: 200px;
  background-color: #ffffff;
  border: 2px solid #333;
  border-radius: 10px;
  padding: 20px;
  margin: 20px;
}
属性 说明 常用值
width/height 设置div的宽度和高度 px, %, auto, vw/vh
background 设置背景颜色或图片 颜色值, url()
border 设置边框样式 solid, dashed, dotted
margin 设置外边距 px, %, auto
padding 设置内边距 px, %
