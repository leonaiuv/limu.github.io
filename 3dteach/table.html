3. 表格的常用属性

table { border-collapse: collapse; width: 100%; border: 1px solid #ddd; } th, td { padding: 8px; text-align: left; border: 1px solid #ddd; } thead { background-color: #f5f5f5; } tbody tr:nth-child(even) { background-color: #f9f9f9; }
属性 说明 常用值
border-collapse 边框合并方式 collapse, separate
border-spacing 单元格间距 0, 2px
caption-side 表格标题位置 top, bottom
empty-cells 空单元格显示 show, hide
