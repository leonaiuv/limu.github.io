表格元素 <table>

表格元素是HTML中用于展示结构化数据的重要组件，通过行和列的形式组织信息，使数据更易于理解和比较。

表格结构与组成

表格由多个元素组成，每个元素都有特定的作用：

<table>

表格的容器元素，包含所有表格内容

<thead>

表格的头部区域，通常包含列标题

<tbody>

表格的主体区域，包含数据行

<tfoot>

表格的底部区域，通常包含汇总信息

<tr>

表格行，包含单元格

<th>

表头单元格，用于定义列标题

<td>

数据单元格，包含表格数据

交互式结构演示

高级表格技巧

单元格合并

使用 colspan 和 rowspan 属性可以合并单元格：

<table border="1">
  <tr>
    <th colspan="2">产品信息</th>
    <th rowspan="2">评价</th>
  </tr>
  <tr>
    <th>名称</th>
    <th>价格</th>
  </tr>
  <tr>
    <td>智能手机</td>
    <td>¥3999</td>
    <td>很好</td>
  </tr>
</table>

响应式表格

使用CSS创建在移动设备上也能良好显示的响应式表格：

/* 响应式表格CSS */
@media screen and (max-width: 600px) {
  table.responsive {
    border: 0;
  }
  
  table.responsive thead {
    display: none;
  }
  
  table.responsive tr {
    margin-bottom: 10px;
    display: block;
    border: 1px solid #ddd;
  }
  
  table.responsive td {
    display: block;
    text-align: right;
    border-bottom: 1px dotted #ccc;
  }
  
  table.responsive td:last-child {
    border-bottom: 0;
  }
  
  table.responsive td:before {
    content: attr(data-label);
    float: left;
    font-weight: bold;
  }
}
