认识SPAN元素

SPAN是HTML中最基本的内联元素，用于在文本中标记特定的部分并应用样式。

3. SPAN的常用属性

span { color: #333333; background-color: #f0f0f0; font-size: 16px; font-weight: normal; font-style: normal; text-decoration: none; padding: 2px 5px; border-radius: 3px; }
属性 说明 示例值
color 文字颜色 #333, red, rgb(0,0,0)
background-color 背景颜色 #f0f0f0, transparent
font-size 字体大小 16px, 1em, 1rem
font-weight 字体粗细 normal, bold, 400
