HTML基础

HTML（超文本标记语言）是网页的基础，用于构建网页的结构。

关键概念：

  • HTML文档结构
  • 语义化标签
  • 表单和输入
  • 多媒体元素

示例代码：

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>页面标题</title>
</head>
<body>
    <h1>Hello World</h1>
</body>
</html>

CSS基础

CSS（层叠样式表）用于设计网页的视觉表现，控制布局、颜色、字体等样式。

核心概念：

  • 选择器和优先级
  • 盒模型
  • 布局方式（Flexbox/Grid）
  • 响应式设计

示例代码：

.container {
    max-width: 1200px;
    margin: 0 auto;
    display: flex;
    gap: 20px;
}

JavaScript基础

JavaScript是一种动态编程语言，用于创建交互式网页效果和Web应用程序。

重要概念：

  • 变量和数据类型
  • 函数和作用域
  • DOM操作
  • 异步编程

示例代码：

// 基本函数示例

    return `Hello, ${name}!`;
}
